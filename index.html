Projects

Brigham and Women's Pathfinding Application

I worked in an agile team of nine WPI students to create a pathfinding application for Brigham and Women's hospital. This application included a pathfinding screen, multiple requests for services (such as spill cleanup and requesting a translator), and a simple room booking service. In the creation of this application, I acted both as a software engineer and the project manager, juggling the writing of features in the application while coordinating and motivating our team. This project used Java, C++, JavaFX, and JFeonix Libraries. This project also strengthed our knowledge on IntelliJ IDE and GitHub.

Escape

I designed the back end of a board game with many different variations. For example, Escape allows for square and hexagonal boards, different piece types with different movement and jumping capabilities, and different sets to rules for game completion. This game was written in Java.

Generic Piece Validator

This is a program that determines is a chess piece is eligible to move from one square to another. This program does not actually simulate the game of chess, just checks to see if moves on a given chess board are legal or not.

Redesigning a Health Center in Dwenase, Ghana

I collaborated with three other WPI students, citizens in a rural Ghanaian village, and the health workers in the aforementioned village to recommend improvements to their local health center. This experience helped me learn how to work with people who are different from me in an environment that I am unfamiliar with.

Sarcasm Detector

I worked with three other WPI students to analyze a data set of sarcastic and serious news headlines. We analyzed this data to determine which words were most common in both categories, which phrases were most common in both categories, and which words were the strongest determiners of sarcasm or seriousness. Lastly, we created a web application that uses a neural network that allows a user to enter in a news headline and determine if our model believes the article to be sarcastic or serious.
